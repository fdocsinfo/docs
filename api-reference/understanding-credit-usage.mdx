## **Personal Data Endpoints**

### 🔹 **Get Personal Profile**

- **Basic call**: `1 credit` for retrieving essential profile details.
- **Additional options** (e.g., skills, certifications, etc.): `+0.5 credits` per option.
- **Maximum: **3 credits.

  **Examples:**
  - Fetching just the main details → `1 credit`
  - Fetching main details \+ certifications \+ skills → `2 credits`
  - Fetching main details \+ skills → `1 credit` _(rounded from 1.5 credits)_

### 🔹 **Other Personal Data Endpoints**

- **Get Profile’s Posts**: `2 credits per call`
- **Detect Activity Time**: `2 credits per call`
- **Search Posts**: `2 credits per call`

### ✅ **Key Takeaways**
